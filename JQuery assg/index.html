Q2: How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

Ans : 

            to apply CSS in jquery : 

            $("#box").css({
                backgroundColor: "black",
                color: "white",
                width: "400px",
                border: "1px solid black",
              });
            
                to add or remove class in jquery : 

                $(document).ready(function () {
                    
                
                    $("#button1").click(function () {               // Add Class
                        $("h1").addClass("active");
                });
                
                $("#button1").click(function () {            // Remove class
                  $("h1").removeClass("active");
                });

            });
        
            to animate in jquery : 
            
            $("#box").css({                                 // giving css to a box div
                backgroundColor: "black",
                color: "white",
                width: "400px",
                border: "1px solid black",
              });

            
            $(document).ready(function () {
                
                $("#btnAnimate").click(function () {            //animation
                    $("#box").animate({
                        
                        width: '200px',
                        
                    },2000)
                });
