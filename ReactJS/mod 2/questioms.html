What is JavaScript Output method?
  1. innerHTML property
  2. innerText property
  3. console.log method
  4. document.write method
  5. window.alert method
  6. window.prompt method
How to JavaScript Output method? 
 for innerHTML : 
 - take input tag, and using dom get it's value say "value1"
 - then take one span tag and give id "spanid" with dom and we can write the syntax
   value1.innerHTML = spanid

 for innerText : 
 - take input tag, and using dom get it's value say "value1"
 - then take one span tag and give id "spanid" with dom and we can write the syntax
   value1.innerText = spanid

 for console.log : 
 - take input tag, and using dom get it's value say "value1"
 - then we can write the syntax
   console.log(value1)

 for document.write : 
 - take input tag, and using dom get it's value say "value1"
 - then we can write the syntax
   document.write(value1)

 for window.alert : 
 - we can write the syntax
   window.alert("we can write anything here")

 for window.prompt : 
 - we can write the syntax
   let a = window.prompt("Enter your name")
   console.log(a);          // name goes to console
