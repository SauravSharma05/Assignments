Q7: What is the difference between Bootstrap 4 and Bootstrap 5

Ans : 

        a : bootstrap 4 
            - it has 5 levels : xs, sm, md, lg, xl - 
            - it includes Jquery plugins- 
            - it has very few colors- 
            - Columns can be positioned relative to one another- 
        
        b :  bootstrap 5 
            - it has 6 levels : xs, sm, md, lg, xl, xxl - 
            - it does not includes Jquery plugins- 
            - it has more colors variations- 
            - Columns cannot positioned relative to one another-
